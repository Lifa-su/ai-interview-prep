# 🎯 AI 面试准备助手

智能面试准备工具，基于 DeepSeek API 提供 AI 生成面试题、模拟面试、答案评估等功能。

## 功能

- **职位选择** — 前端、后端、全栈、产品、设计
- **难度分级** — 初级 / 中级 / 高级
- **AI 生成面试题** — 技术题 + 行为题，支持公司定向准备
- **模拟面试** — 逐题展示，计时作答，AI 实时评估
- **AI 评估反馈** — 评分、优点、改进建议、参考答案
- **内置题库** — 50+ 常见面试题，按职位分类
- **答题历史** — 本地存储，进度追踪，统计分析

## 使用

1. 直接用浏览器打开 `index.html`
2. 在设置页填入 [DeepSeek API Key](https://platform.deepseek.com)
3. 选择职位类型和难度级别
4. 开始练习或模拟面试

## 技术栈

- 纯前端：单文件 HTML + CSS + JS
- AI：DeepSeek API (`deepseek-chat`)
- 存储：localStorage
- 零依赖，无需构建
